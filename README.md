# 📱 Android Smartphone Sales Dashboard (Power BI)

## 📌 Overview

This project is an interactive Power BI dashboard designed to analyze Android smartphone sales data across different regions, brands, and time periods. It provides insights into revenue, customer behavior, and sales performance.

---

## 🎯 Objective

* Analyze overall sales performance
* Track quantity and transaction trends
* Understand customer purchasing behavior
* Identify top-performing brands and models

---

## 📊 Key Features

* Total Sales, Quantity, Transactions, Average Price KPIs
* Region-wise sales visualization (map)
* Monthly sales trend analysis
* Top-selling smartphone models
* Payment method distribution
* Customer rating analysis
* Day-wise sales trends
* Brand-wise performance comparison

---

## 📈 Key Insights

* Strong sales concentration in major cities
* Balanced payment distribution across UPI, cards, and cash
* Consistent monthly sales with peak periods
* Top brands contributing majority of revenue

---

## 🛠️ Tools Used

* Power BI
* Data Cleaning & Transformation
* Data Visualization
